==============
Transcriptions
==============

Dymos implements several different *transcriptions* which are used to convert a continuous
optimal control problem into an analogous nonlinear programming (NLP) problem. Currently,
Dymos supports the following techniques:

.. toctree::
    :maxdepth: 2
    :titlesonly:

    gauss-lobatto
    radau-pseudospectral
    runge-kutta
